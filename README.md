##  Udacity Statistics for Data Analysis Nanodegree

This repository contains the four projects completed in the Statistics for Data Analysis Nanodegree

## Project 1 - Describe Health and Sleep Quality Data

### Project Scenario

Gaining an understanding of your data is the first step in any data analysis project.

For this project, you will be describing data related to health and sleep quality.

You will use the software of your choice to calculate descriptive statistics about this dataset, then report your findings in a slide deck presentation.

## Project 2 - Calculation of Basketball Scoring Probabilities

### Project Scenario

Probability helps you make sense of uncertain situations.

For this project, you will be analyzing data related to basketball shooting statistics for NBA player Steph Curry.

You will use the software of your choice to calculate probabilities related to this dataset, then report your findings in a slide deck presentation.

## Project 3 - Analyze A-B Test Results

### Project Scenario

A/B tests are very commonly performed by data analysts and data scientists. It is important that you get some practice working with the difficulties of these.

For this project, you will be working to understand the results of an A/B test run by an e-commerce website. The company has developed a new web page in order to try and increase the number of users who "convert," meaning the number of users who decide to pay for the company's product.

## Project 4 - Model Salary Data With Linear Regression

### Project Scenario

In this project, you will build a linear model and interpret the results of the model. Specifically, you will look at what variables are related to salaries.
